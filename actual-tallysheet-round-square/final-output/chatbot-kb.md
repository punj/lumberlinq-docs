# Chatbot Knowledge Base: Tally Sheet Round and Square

Intent: `explain_tallysheet`

Answer: Tally Sheets in LumberLinq record timber measurements against a transport unit. Round tally sheets use Length and Girth. Square tally sheets use Width, Thickness, Length, and Pieces.

Intent: `round_tallysheet`

Answer: A Round tally sheet is used for round logs. Enter Length and Girth, then LumberLinq calculates gross and net CBM/CFT values.

Intent: `square_tallysheet`

Answer: A Square tally sheet is used for sawn or rectangular timber. Enter Width, Thickness, Length, and Pieces. LumberLinq calculates volume totals automatically.

Intent: `tallysheet_settings`

Answer: Open Settings in the tally grid to manage validation rules, unit of measure, allowances, decimal precision, visible columns, Copy Previous Length, and Add to Length behavior.

Intent: `tallysheet_export`

Answer: Use Export from the tally toolbar. The verified menu includes Excel, PDF, Bundle, and Advanced options.

Intent: `tallysheet_import`

Answer: Use Import from the tally toolbar for file-based import. The import flow supports configuration review, upload and mapping, preview, and importing rows into the grid before saving.

Intent: `tallysheet_ai_import`

Answer: Use AI Import from the tally toolbar for image-based extraction of handwritten tally sheets. The AI Import component supports image upload, AI credit display, extraction, preview, and import into the tally grid.

Intent: `tallysheet_status_bar`

Answer: The status bar shows total rows, gross CBM/CFT, unsaved row count, save progress, saved state, and ready state.

Intent: `tallysheet_row_actions`

Answer: Row actions include Add Row Below and Delete Row. Copy Previous Length is controlled from Settings and can copy the previous length when adding rows.

Intent: `tallysheet_validation`

Answer: Validation highlights incomplete or invalid entries. Round validation applies to Length and Girth. Square validation applies to Width, Thickness, Length, and Pieces.
