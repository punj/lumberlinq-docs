# YouTube Description

Learn how to manage timber shipments in LumberLinq from shipment creation through inventory, payments, export, and reconciliation.

This walkthrough covers the Shipments list, search and filters, shipment tabs, party and consignment details, linked transport units, shipment view, documents, payment tracking, export options, inventory overview, inventory movement ledger, adjustments, processing runs, reconciliation reports, inventory reports, share links, and shipment locking.

Covered in this video:

- Shipment list, search, and filters
- Party and consignment details
- Transport units linked to shipment
- Shipment view with transport-unit tabs
- Documents and shipment file controls
- Payment summary and record-payment form
- Export with linked transport units
- Inventory overview and In/Out ledger
- Inventory adjustments and processing runs
- Reconciliation and inventory reports
- Share links, lock control, and mobile view
