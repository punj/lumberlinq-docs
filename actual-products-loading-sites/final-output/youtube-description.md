# YouTube Description

Learn how to manage Products and Loading Sites in LumberLinq.

This tutorial shows how timber and logistics teams can maintain product catalog records, search product lists, create and edit timber products, manage loading yards and operational sites, and use validation and confirmation steps to keep master data clean.

Covered in this video:
- Products list and search
- Creating timber product records
- ROUND and SQUARE product formulas
- Product validation and delete confirmation
- Loading Sites list and search
- Creating and editing loading sites
- Loading site validation and delete confirmation
- Practical master data tips for timber operations

LumberLinq helps timber exporters, importers, traders, logistics teams, and yard supervisors reduce spreadsheet dependency, improve visibility, and run faster operations from centralized records.
