# YouTube Description

Learn how to use key LumberLinq modules: Dashboard V5, Inventory, Transport Units, Utilities, Subscriptions, Storage, and Support Tickets.

This tutorial covers:
- Dashboard V5 business overview
- Inventory overview, movements, adjustments, and processing
- Transport unit workflow from tally sheet to shipment
- Unit conversion, volume estimates, and slab generator
- Subscription plans, plan limits, and payment history
- Storage usage and file list
- Support ticket creation and tracking

LumberLinq is built for timber exporters, importers, traders, logistics teams, and yard supervisors.
