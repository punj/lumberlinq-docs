# YouTube Description

Learn how to use LumberLinq Reports, Manage User, and Role-Based Access Control for timber and logistics operations.

This tutorial covers report filters, Excel/PDF exports, shipment reporting, financial reporting, inventory reporting, user invitations, pending invitations, RBAC settings, role templates, and access-rights configuration.

Best for timber exporters, importers, traders, yard teams, finance teams, and logistics managers who want cleaner reporting and controlled user access.
