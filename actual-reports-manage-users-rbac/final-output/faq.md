# FAQ - Reports, Manage User, and RBAC

## Reports

**Q: Which report should I use for shipment performance?**
A: Use the Shipment Report. It includes shipment totals, CBM, pieces, trade type, parties, ports, BL number, and financial status.

**Q: Which report should I use for receivables and payables?**
A: Use the Financial Report. It includes actual received, actual paid, receivable, payable, overdue, partner, currency, trend, and transaction views.

**Q: Which report should I use for stock and processing?**
A: Use the Inventory Report. It shows available stock, in-process stock, shipment stock, movements, adjustments, and processing runs.

**Q: Can I export reports?**
A: Yes. Supported reports show Excel and PDF export buttons.

**Q: Why is a report empty?**
A: Check the date range and filters, then click Generate or Refresh.

## Manage User

**Q: How do I invite a user?**
A: Open Manage User, click Invite User, enter one or more email addresses, choose a role template if RBAC is enabled, and send the invite.

**Q: Can I invite multiple users at once?**
A: Yes. Separate email addresses with commas, spaces, or new lines.

**Q: What happens if an email is invalid?**
A: LumberLinq shows a validation message before the invite is sent.

**Q: Where can I see pending invitations?**
A: Pending invitations appear below the user table on the Manage User page.

## RBAC

**Q: What is RBAC?**
A: RBAC is Role-Based Access Control. It lets administrators control which modules and actions each user can access.

**Q: What role templates are available?**
A: Field Operator, Supervisor, Analyst, Manager, and Custom.

**Q: Can permissions be changed after inviting a user?**
A: Yes. Open the user Access Rights panel and adjust permissions.

**Q: Can I enable all permissions for one module?**
A: Yes. Each permission group supports Enable all and Disable all actions.

**Q: What happens if RBAC is disabled?**
A: Users return to broader role-based access, while saved permission settings are preserved for later reactivation.
