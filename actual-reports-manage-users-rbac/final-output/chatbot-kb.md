# Chatbot Knowledge Base - Reports, Manage User, and RBAC

## Reports Overview
LumberLinq Reports provide operational and financial views for timber logistics. Available reports include Business Partner, Product, Loading Site, Transport Unit, Tally, Reconciliation, Shipment, Financial, Inventory, Users, and App Usage.

## How to Generate a Report
Tell the user to open Reports, select the relevant report, set filters such as date range, trade type, status, currency, or movement type if available, then click Generate or Refresh.

## Report Exports
If the user asks how to download a report, explain that supported report screens have Excel and PDF export buttons in the report header.

## Shipment Report
Use Shipment Report for shipment counts, volume, pieces, trade type, parties, ports, BL numbers, payment status, transactions, and financial shipment details.

## Financial Report
Use Financial Report for receivables, payables, overdue amounts, actual received, actual paid, currency totals, partner balances, monthly trends, and transaction details.

## Inventory Report
Use Inventory Report for available stock, in-process stock, in-shipment stock, shipped-this-month metrics, movement log, adjustments, and processing runs.

## Manage User
Manage User shows team members, role, active/inactive status, login type, last active time, and pending invitations. Administrators can invite users, deactivate/reactivate users, manage invitations, and open Access Rights.

## Invite Users
Users can be invited with one or more email addresses. Multiple emails can be separated by commas, spaces, or new lines. Invalid emails show validation errors. If RBAC is enabled, a role template can be selected before sending.

## RBAC Settings
RBAC lets administrators configure per-user access. The RBAC page shows enabled/disabled status, total users, configured users, no-access users, average coverage, permission counts, and last-modified details.

## Access Rights
The Access Rights sidebar supports role templates, module permission groups, individual permission toggles, Enable all, Disable all, and Save Changes.

## Permission Modules
Permission modules include Dashboard, Tally Sheet, Shipments, Business Partner, Reports, Storage, and Users & Company.

## Suggested Chatbot Answers
- For report export: "Open the report and use the Excel or PDF button in the report header."
- For user invitation: "Open Manage User, click Invite User, enter email addresses, select a role template if RBAC is enabled, then send the invite."
- For restricting a user: "Open RBAC settings or Manage User, click the shield/configure action, adjust permissions in Access Rights, and save."
- For pending invites: "Open Manage User and check the Pending Invitations section below the user table."
